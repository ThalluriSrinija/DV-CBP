# E-Commerce Customer Segmentation Dashboard

## 📊 Project Overview
This project analyzes customer behavior using RFM analysis and K-Means clustering.

## 🚀 Features
- Customer segmentation (RFM)
- Interactive dashboard (Streamlit)
- Filtering by country & segment
- 14+ visualizations

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

## ▶️ Run Project
```bash
streamlit run dashboard.py
